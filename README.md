# list-leyu-hub

本仓库用于归档和发布多个独立的 HTML 页面，主要作为个人前端资源或静态页面的集合。

## 项目简介

list-leyu-hub 是一个简单的静态页面归档仓库，收录了若干独立制作的 HTML 文件。每个页面均独立完整，不依赖特定后端服务，适合直接通过浏览器打开或部署至静态托管平台。

仓库内容不针对任何特定网站或域名，仅作为前端练习、工具页面或信息展示页面的统一存放地。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各个独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库入口页面，可提供页面列表或简单导航。
- `pages/`：各独立 HTML 页面存放目录，每个文件为一个完整页面。
- `assets/`：存放页面共享的样式、脚本或图片资源。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开使用。
- 页面间无强依赖关系，可单独部署或引用。
- 资源文件尽量使用相对路径，便于整体迁移。

## 维护说明

- 本仓库不定期更新，新增或修改页面会直接提交到主分支。
